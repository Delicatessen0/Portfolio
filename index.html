Featured Project

Steam Game Success Predictor

An end-to-end ML pipeline that pulls live data from the Steam API to predict a game's success — measured by peak concurrent players and review score. Combines NLP-based sentiment analysis on user reviews with tabular feature engineering (genre, price, developer history, tag embeddings) to train a gradient-boosted classifier.

  • Python
  • Steam API
  • XGBoost
  • NLP / VADER
  • Matplotlib
