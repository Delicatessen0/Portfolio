Some Things I've Built

Featured Project

Game Recommendation LLM

A recommendation engine that matches players with games they'll actually enjoy. I used HuggingFace transformers to generate semantic embeddings of game descriptions, and built a FAISS vector index to run fast content-based queries. It was a fun way to explore how modern NLP can improve game discovery.

  • Python
  • HuggingFace
  • Transformers
  • SQL
  • FAISS

Featured Project

Steam Game Success Predictor

An end-to-end machine learning pipeline that uses XGBoost to predict if a game will be a hit or a flop. I fetched live telemetry and metadata (concurrent users, pricing, developer history, user tags) from the SteamSpy API, ran VADER sentiment analysis on descriptions, and trained a classifier to identify success indicators with 85.7% accuracy.

  • Python
  • Steam API
  • XGBoost
  • NLP / VADER
  • Matplotlib

Featured Project

Player Churn Prediction

A survival analysis and classification model built to identify when live-service players are about to quit. By looking at telemetry data like play frequency, session length, and in-game progression, the model predicts churn risk so developers can proactively design better retention strategies.

  • Python
  • XGBoost
  • Scikit-Learn
  • Pandas
  • Data Viz
